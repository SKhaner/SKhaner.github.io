LC3 Cipher

In this LC3 project I implement a simple ARX cipher (with a rotational component replaced by a shift operation) encryption algorithm which we call "SAD" based on a composition of three methods - Caeser's cipher, the Vigenere Cipher, and a right/left shift. The goal is to encrypt a message, consisting of ASCII characters, using a key provided by the user. To decrypt the message correctly, the same key must be used by the decryption algorithm.
